مميزات قوية

كل ما تحتاجه لتحميل وإدارة الفيديوهات من الإنترنت.

جودة تصل إلى 4K

حمّل الفيديوهات بجودات متعددة تشمل Ultra HD 4K مع دمج تلقائي للstreams.

دعم قوائم التشغيل

الصق رابط قائمة التشغيل وحمّل جميع الفيديوهات دفعة واحدة.

قائمة انتظار التحميل

أضف عدة تحميلات من إضافة المتصفح وستُحمّل واحداً تلو الآخر تلقائياً.

إيقاف وإلغاء

تحكم كامل في التحميلات — أوقف مؤقتاً أو أعد التشغيل أو ألغِ بضغطة واحدة.

إضافة المتصفح

إضافة بأسلوب IDM لـ Chrome و Edge و Firefox. كشف وتحميل الفيديوهات بنقرة واحدة.

تشغيل تلقائي عند الإقلاع

يعمل في شريط المهام تلقائياً عند تسجيل الدخول. جاهز دائماً لالتقاط التحميلات.

تقدم مباشر

عرض فوري لسرعة التحميل والوقت المتبقي وحجم الملف ونسبة التقدم.

قابل للتخصيص

اضبط صيغة التصدير (MP4/MKV/WebM) والأجزاء المتوازية والمحاولات والمزيد.

وضع مظلم

واجهة جميلة بلون مظلم مع لمسة كهرمانية. مريحة للاستخدام نهاراً وليلاً.

كيفية الاستخدام

ابدأ في دقائق مع خيارات تثبيت متعددة.

تثبيت سريع (pip)

  1. 1
    ثبّت Python 3.10+ و FFmpeg
    # ويندوز
    winget install Gyan.FFmpeg

    # macOS
    brew install ffmpeg

    # Linux
    sudo apt install ffmpeg
  2. 2
    ثبّت YTGrab وأشغله
  3. 3
    الصق رابط يوتيوب واضغط Fetch
  4. 4
    اختر الجودة واضغط Start Download
pip install git+https://github.com/a7mdmo74/ytgrap.git
ytgrab

إضافة المتصفح

  1. 1
    حمّل وثبّت YTGrab (أو شغّله من المصدر)
  2. 2
    افتح متصفحك واذهب إلى chrome://extensions
  3. 3
    فعّل وضع المطور (الزر في الأعلى يمين)
  4. 4
    اضغط Load unpacked واختر مجلد browser_extension
  5. 5
    تصفح أي منصة مدعومة — يظهر زر التحميل على الفيديوهات
