How to Use

Get started in minutes with multiple installation options.

Quick Install (pip)

  1. 1
    Install Python 3.10+ and FFmpeg
    # Windows
    winget install Gyan.FFmpeg

    # macOS
    brew install ffmpeg

    # Linux
    sudo apt install ffmpeg
  2. 2
    Install and run YTGrab
  3. 3
    Paste a YouTube URL and click Fetch
  4. 4
    Select quality and click Start Download
pip install git+https://github.com/a7mdmo74/ytgrap.git
ytgrab

Browser Extension

  1. 1
    Download and install YTGrab (or run from source)
  2. 2
    Open your browser and go to chrome://extensions
  3. 3
    Enable Developer mode (toggle in top right)
  4. 4
    Click Load unpacked and select the browser_extension folder
  5. 5
    Browse any supported platform — download button appears on videos
