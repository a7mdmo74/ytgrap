# YTGrab

مدير تحميل مقاطع يوتيوب مع دعم متصفح بأسلوب IDM.

يعمل على **ويندوز** و**ماك** و**لينكس**.

## المميزات

- تحميل مقاطع يوتيوب بجودات متعددة (حتى 4K)
- دعم قوائم التشغيل — يكتشف ويحمّل جميعمقاطع تلقائياً
- عرض التقدم والسرعة والوقت المتبقي وحجم الملف مباشرة
- إعدادات قابلة للتعديل: صيغة الإخراج (MP4/MKV/WebM) وعدد القطع المتوازية وعدد المحاولات
- دمج تلقائي للفيديو والصوت
- ثيم داكن مع لون كهرماني (بأسلوب IDM)

### جديد: إضافة المتصفح (مثل IDM!)

- ثبّت إضافة كروم لاكتشافمقاطع تلقائياً
- تصفّح يوتيوب وانستغرام وتiktok وتويتر/إكس وفيسبوك وفيميو والمزيد
- اضغط زر التحميل الذي يظهر على أي مقطع
- تُضاف المقاطع إلى قائمة الانتظار في التطبيق ويُحمّل تلقائياً
- يعمل مع أكثر من 12 منصة تواصل اجتماعي
- صفحة إعدادات قابلة للتعديل

## المتطلبات

- **Python 3.10+**
- **FFmpeg**

### تثبيت FFmpeg

| المنصة | الأمر |
|--------|-------|
| ويندوز | `winget install Gyan.FFmpeg` |
| ماك | `brew install ffmpeg` |
| لينكس | `sudo apt install ffmpeg` |

## التثبيت

### الخيار 1: التشغيل من المصدر (جميع المنصات)

```bash
git clone https://github.com/a7mdmo74/ytgrap.git
cd ytgrap
pip install -r requirements.txt
python youtube_downloader.py
```

### الخيار 2: التثبيت عبر pip

```bash
pip install git+https://github.com/a7mdmo74/ytgrap.git
ytgrab
```

### الخيار 3: مُثبّت ويندوز

حمّل أحدث إصدار من [إصدارات GitHub](https://github.com/a7mdmo74/ytgrap/releases)، واستخرج الملفات، وشغّل `install.bat` كمسؤول.

## إعداد إضافة المتصفح

### كروم / إيدج / برايف

1. افتح المتصفح وانتقل إلى `chrome://extensions`
2.فعّل **وضع المطوّر** (زر التبديل في أعلى اليمين)
3. اضغط **تحميل مفكوك**
4. اختر مجلد `browser_extension` من هذا المشروع
5. سيظهر أيقونة YTGrab في شريط الأدوات

### فايرفوكس

1. افتح فايرفوكس وانتقل إلى `about:debugging#/runtime/this-firefox`
2. اضغط **تحميل إضافة مؤقتة**
3. اختر أي ملف في مجلد `browser_extension`

### كيفية العمل

1. شغّل تطبيق YTGrab (يبدأ الخادم المحلي تلقائياً على البورت 19850)
2. تصفّح يوتيوب أو انستغرام أو تiktok أو أي منصة مدعومة
3. اضغط زر "تحميل" الذي يظهر على المقاطع
4. يُضاف المقطع إلى قائمة الانتظار في التطبيق
5. اضغط زر التشغيل (▶) لبدء التحميل

### إعدادات الإضافة

انقر بالزر الأيمن على أيقونة YTGrab في شريط الأدوات واختر **خيارات** للإعداد:

**اتصال الخادم**
- تعيين عنوان خادم مخصص (الافتراضي: `http://127.0.0.1:19850`)
- اختبار الاتصال للتحقق من تشغيل التطبيق

**سلوك التحميل**
- تحميل تلقائي عند الاكتشاف — يبدأ التحميل تلقائياً عند اكتشاف مقطع
- إظهار زر التحميل — تبديل ظهور زر التحميل على المقاطع
- إظهار اللوحة العائمة — تبديل لوحة YTGrab على صفحات المقاطع
- إظهار الإشعارات — تبديل إشعارات التنبيه

**المنصات المدعومة**
- تفعيل/تعطيل الاكتشاف لكل منصة على حدة
- يوتيوب وانستغرام وفيسبوك وتويتر/إكس وتiktok وفيميو وديلي موشن وتويتش وريديت وبيلي比ت وساوند كلاود

### المنصات المدعومة

- يوتيوب، يوتيوب ميوزك
- انستغرام (ريلز، ستوريز، منشورات)
- فيسبوك (مقاطع، ريلز)
- تويتر/إكس (مقاطع)
- تiktok
- فيميو
- ديلي موشن
- تويتش (VOD)
- ريديت (مقاطع)
- بيلي比ت
- ساوند كلاود

## طريقة الاستخدام

### التحميل اليدوي

1. الصق رابط مقطع أو قائمة تشغيل يوتيوب
2. اضغط **جلب** لتحميل الجودات المتاحة
3. اختر الجودة من القائمة المنسدلة
4. اختر مكان الحفظ (الافتراضي: التنزيلات)
5. اضغط **بدء التحميل**

### التحميل عبر إضافة المتصفح

1. ثبّت إضافة المتصفح (انظر أعلاه)
2. شغّل تطبيق YTGrab
3. تصفّح أي منصة مدعومة
4. اضغط زر التحميل على المقاطع
5. راقب التقدم في قائمة انتظار المتصفح

## إعدادات التطبيق

- **صيغة الإخراج**: MP4 أو MKV أو WebM
- **القطع المتوازية**: 1-8 (الافتراضي: 3)
- **المحاولات القصوى**: 1-30 (الافتراضي: 10)
- ** التنظيف التلقائي**: حذف ملفات .part/.temp المتبقية
